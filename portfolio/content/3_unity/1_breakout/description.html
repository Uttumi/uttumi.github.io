A typical Breakout/Arkanoid clone. How elementary of me.

I've been teaching game development for a few years by now and this is always the project I start with. That means a lot of Breakout games out there. It's one of the best games to show some basics of Unity and game programming in general. It has simple controls on one axis, some physics you could easily even write on your own and a little potential for animations.

So it's nothing special, but I wanted to give you a chance to beat the level.
